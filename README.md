# MBC 아카데미 학사 관리 시스템
학원 사무실에서 학생과 강사에 대한 서류와 사전 평가 등 모든 업무를 서면으로 작성하는 것을 보게 되었습니다. 학사 관리 시스템이 있으면 좋지 않겠냐고 제안을 드렸고, 팀원들을 모아 프로젝트를 시작하게 되었습니다.

[Debugging Dragons]
조장: 임새별
조원: 윤지영, 박혜연, 김지혜


# 📌 기능

김지혜(wisdom): 상담
